# Marker

___Powerful markdown editor for the GNOME desktop___

Features:

- View and edit markdown documents
- Convert markdown documents to HTML
- Tex math rendering
- Support for mermaid diagrams
- Support for charter plots
- Syntax highlighting for code blocks
- Integrated sketch editor
- Exports to PDF, RTF, ODT, DOCx, LaTeX
- Custom CSS themes
- Custom syntax themes
- Figure captioning and numbering

---

**Manual Install**

Make sure you follow the [setup guide for your Linux distribution](https://flathub.org/en/setup) before installing

```
flatpak install flathub com.github.fabiocolacio.marker
```

**Run**

```
flatpak run com.github.fabiocolacio.marker
```

---

**Technologies**: GNOME, GTK, webkit2gtk-4, gtksourceview-3, C
